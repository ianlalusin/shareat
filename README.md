# SharEat Hub (shareat-v2)

This is the repository for the SharEat Hub application, a comprehensive Point-of-Sale (POS), Kitchen Display System (KDS), and Enterprise Resource Planning (ERP) solution for restaurants.

## Project Overview

This application is built with Next.js, React, TypeScript, and Firebase. It provides a full suite of tools for managing restaurant operations, including:

-   **Multi-Store Management**: Centralized administration for multiple restaurant locations.
-   **Role-Based Access Control**: Granular permissions for Admins, Managers, Cashiers, Servers, and Kitchen staff.
-   **Point of Sale (POS)**: A feature-rich cashier interface for starting sessions, managing bills, applying discounts, and processing payments.
-   **Kitchen Display System (KDS)**: A real-time order tracking system for kitchen staff to manage food preparation.
-   **Server Station**: A dedicated view for servers to verify guest counts and manage tables.
-   **Inventory & Menu Management**: Global and store-level control over products, packages, flavors, and pricing.

## Technology Stack

-   **Frontend**: Next.js with React & TypeScript
-   **UI**: ShadCN UI, Tailwind CSS, Radix UI
-   **Backend & Database**: Firebase (Firestore, Firebase Authentication, Firebase Storage)
-   **Generative AI**: Genkit

## Getting Started

To run this project locally, you will need to have Node.js and npm installed.

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run the Development Server**:
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
